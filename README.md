# Sraddha Kanuparthy — Portfolio

A modern, professional developer portfolio website for a Computer Science student.

## Features

- Blue, white, and dark navy developer theme
- Glassmorphism cards and smooth scroll animations
- Fully responsive (desktop & mobile)
- Sections: Hero, About, Skills, Projects, Certifications, Achievements, Contact, Footer

## Quick Start

1. Open `index.html` in any modern browser, or
2. Use a local server (recommended):

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

Then visit `http://localhost:8080`
